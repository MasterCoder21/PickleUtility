Fun Games

We are still working on collecting web-friendly games. Here are a few that you might like!

Slope

An endless game where the goal is to keep the ball on ramps. Make sure to avoid the red parts!

2048

Match numbered blocks and get the 2048 block! A challenging but fun and addicting game.

987

In this variation of 2048, you are matching numbers from the Fibonacci sequence with the goal of getting the 987 tile.

Syobon Action

Cat Mario, but you have to remember where to go, because everything hurts you. Can you survive the madness?
